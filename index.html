Olá! Sou Graziele Santos, estudante de Desenvolvimento de Sistemas com foco em criar soluções tecnológicas inovadoras. Estou aprendendo as mais recentes tecnologias e metodologias para desenvolver aplicações web e mobile que resolvam problemas reais.

Atualmente, estou aprimorando minhas habilidades em HTML, CSS, JavaScript e frameworks modernos como React. Também tenho interesse em desenvolvimento backend com Node.js e bancos de dados.
